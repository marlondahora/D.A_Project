Sustainable Development Goals

In September 2015, the United Nations (UN) created a plan to tackle the economic, social and environmental indicators issues. This decision was made based on reports of worsening global stability. This instability had the potential to compromise the futures of generations to come. The UN proposed that its 193 member countries sign the 2030 agenda, a global plan composed of 17 interconnected goals called SDGs. There are 169 objectives for these countries to be achieved by 2030. There are 230 indicators to measure the SDGs progress. Further information

Shoreline

This project is a final year assignment which explores, performs an analysis of, and develops an interactive dashboard based on SDGs progress. The application was built in R studio using several packages and hosted in Shiny server. The dashboard contains time-series analysis, normality tests and many plots for visual interpretation. Also, a prediction of the possible future outcome is based on past data, using the machine learning in R library called forecast (auto.arima).
